## 📊 案例分析考试重点

### 考试题型分布
1. **计算题**（必考，25分左右）
   - 挣值管理（EV、AC、PV、CPI、SPI等）
   - 网络图（关键路径、总时差、自由时差）

2. **问答题**（找错、改错、改进提升）
   - 识别项目管理中的问题
   - 提出改进措施

3. **理论题**（概念、流程、工具）
   - 管理计划内容
   - 过程输入输出
   - 工具与技术

4. **客观题**（选择、填空、判断）
   - 基础概念
   - 常识性知识

### 备考优先级
```
第1梯队：计算题（必须100%掌握）
第2梯队：找问题解决问题（核心得分点）
第3梯队：理论题（选择性背诵）
第4梯队：客观题（日常积累）
```

## 💡 使用建议

1. **循序渐进**：按照目录顺序学习，不要跳跃
2. **反复练习**：案例分析重在练习，多做多总结
3. **建立笔记**：整理自己的问题清单和答题模板
4. **模拟考试**：考前一周每天模拟一套完整试题
5. **时间管理**：案例分析90分钟3道题，平均每题30分钟
