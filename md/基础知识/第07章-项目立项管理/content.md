## 7.1 立项管理概述

**定义**：项目立项管理是对拟规划和实施的项目在技术、经济、实施、社会价值等方面进行全面科学的综合分析，为项目决策提供客观依据的一种技术经济研究活动。

**四个阶段**：
1.  **项目建议与立项申请**
2.  **初步可行性研究**
3.  **详细可行性研究**
4.  **评估与决策**

> **注意**：
> *   这四个阶段统称为**项目投资前时期**。
> *   在实际工作中，初步和详细可行性研究可根据规模和繁简程度合并，但**详细可行性研究是不可缺少的**。
> *   升级改造项目只做初步和详细研究。
> *   小项目一般只进行详细可行性研究。

## 7.2 项目建议与立项申请

### 1. 立项申请概念
*   **别名**：项目建议书。
*   **性质**：是项目建设单位向上级主管部门提交的必须文件，是对拟建项目提出的**框架性总体设想**。
*   **作用**：
    *   项目发展周期的初始阶段。
    *   国家或上级主管部门选择项目的依据。
    *   可行性研究的依据。
    *   涉及利用外资的项目，批准后方可开展后续工作。

### 2. 项目建议书核心内容 (背诵)
1.  **项目的必要性**。
2.  **项目的市场预测**。
3.  **项目预期成果**（如产品方案或服务）的市场预测。
4.  **项目建设必需的条件**。

## 7.3 项目可行性研究

**定义**：在项目建议书被批准后，从技术、经济、社会和人员等方面进行调查研究，论证技术方案，确定项目是否可行。

### 7.3.1 可行性研究的内容 (重点)

#### 1. 技术可行性分析
*   **核心问题**：现有的技术、资源能否实现项目目标，能否在规定期限内完成。
*   **考虑因素**：
    1.  项目开发的风险。
    2.  人力资源的有效性。
    3.  技术能力的可能性。
    4.  物资（产品）的可用性。
*   **重要性**：往往决定项目的方向，一旦估计错误将导致严重后果。

#### 2. 经济可行性分析
*   **核心问题**：投资回报分析。
*   **具体内容**：
    1.  **支出分析**：
        *   **一次性支出**：开发费、培训费、差旅费、设备购置费等。
        *   **非一次性支出**：软硬件租金、人员工资福利、水电费、消耗品等。
    2.  **收益分析**：
        *   **直接收益**：销售收入等直接经济效益。
        *   **间接收益**：成本降低等。
        *   **其他收益**：知识产权、著作权等。
    3.  **收益投资比、投资回收期分析**。
    4.  **敏感性分析**：关键因素变化对支出的影响。

#### 3. 社会可行性分析
*   **组织内部**：品牌、竞争力、技术创新、人员提升、管理提升。
*   **组织外部**：公共效益、文化、环境、社会责任感。

#### 4. 运行可行性分析
*   **核心**：评估是否可以建立系统顺利运行所需要的环境。

#### 5. 其他方面的可行性分析
法律、政策、合同责任、知识产权等。

### 7.3.2 初步可行性研究
*   **时机**：在市场或客户情况调查后进行。
*   **目的**：对项目进行初步评估。
*   **特点**：内容与详细可研基本相同，但在资源占有和细节深度上有差异。
*   **辅助（功能）研究**：只涉及项目的一个或几个方面，作为前提或辅助。费用应计入项目可行性研究费用。

### 7.3.3 详细可行性研究
*   **地位**：不可缺少。最终提交的报告是项目评估和决策的依据。
*   **原则**：科学性、客观性、公正性。
*   **方法**：经济评价法、市场预测法、投资估算法、增量净效益法。

**初步可研与详细可研内容对比：**

| 初步可行性研究内容 | 详细可行性研究的内容 |
| :--- | :--- |
| (1) 需求与市场预测 | (1) 市场需求预测 |
| (2) 设备与资源投入分析 | (2) 部件和投入的选择供应 |
| (3) 空间布局 | (3) 信息系统架构及技术方案的确定 |
| (4) 项目设计 | (4) 技术与设备选择 |
| (5) 项目进度安排 | (5) 网络物理布局设计 |
| (6) 项目投资与成本估算 | (6) 投资、成本估算与资金筹措 |
| | (7) 经济评价及综合分析 |

**成本分类补充**：
项目总成本一般划分为：研发成本、行政管理费、销售与分销费用、财务费用和折旧。前三类总和称为**经营成本**。

## 7.4 项目评估与决策

### 1. 项目评估
*   **定义**：在**可行性研究的基础上**，由**第三方**（国家、银行或有关机构）进行的评价、分析和论证过程。
*   **目的**：审查可行性研究的可靠性、真实性和客观性，为决策提供依据。
*   **依据**：
    1.  项目建议书及其批准文件。
    2.  项目可行性研究报告。
    3.  申请报告及初审意见。
    4.  协议文件。
    5.  其他必需文件。
*   **成果**：**项目评估报告**。

### 2. 评估工作程序
1.  成立评估小组。
2.  开展调查研究。
3.  分析与评估。
4.  编写、讨论、修改评估报告。
5.  召开专家论证会。
6.  评估报告定稿并发布。

## 本章练习与真题

**练习题**

(1) 立项申请、初步可行性研究、详细可行性研究、评估与决策是投资前时期的四个阶段，其中（ ）是不可缺少的。
A. 立项申请
B. 初步可行性研究
C. 详细可行性研究
D. 评估与决策
**【参考答案】C**

(2) ( ) 是项目发展周期的初始阶段，是国家或上级主管部门选择项目的依据，也是可行性研究的依据。
A. 项目建议书
B. 项目评估
C. 项目决策
D. 项目规划
**【参考答案】A**


(3) ( ) 是指在当前市场的技术、产品条件限制下，能否利用现在拥有的以及可能拥有的技术能力、产品功能、人力资源来实现项目的目标、功能、性能，能否在规定的时间期限内完成整个项目。
A. 技术可行性研究
B. 经济可行性分析
C. 运行环境可行性分析
D. 投资可行性研究
**【参考答案】A**

(4) 软、硬件租金、人员工资及福利、水电等公用设施使用费属于 ( )。
A. 直接支出
B. 支持型
C. 非一次性支出
D. 项目支出
**【参考答案】C**
*(解析：非一次性支出是指项目支出中的一部分，不属于项目支出的一部分)*

**历年真题**

在可行性报告中，“是否存在人力资源不足的问题，是否可以通过社会招聘或培训获得所需人员”属于（ ）的内容。
