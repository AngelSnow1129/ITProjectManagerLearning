# 常用公式速查表

> 信息系统项目管理师考试常用公式快速查询

## 📊 项目管理计算公式

### 1. 进度管理

#### 三点估算
- **期望工期（PERT）**：`TE = (To + 4Tm + Tp) / 6`
  - To：乐观时间
  - Tm：最可能时间
  - Tp：悲观时间

- **标准差**：`σ = (Tp - To) / 6`

- **方差**：`σ² = [(Tp - To) / 6]²`

#### 关键路径法（CPM）
- **最早开始时间（ES）**：前置活动最早完成时间的最大值
- **最早完成时间（EF）**：`EF = ES + 工期`
- **最迟完成时间（LF）**：后续活动最迟开始时间的最小值
- **最迟开始时间（LS）**：`LS = LF - 工期`
- **总浮动时间（TF）**：`TF = LF - EF = LS - ES`
- **自由浮动时间（FF）**：`FF = 紧后活动ES的最小值 - 本活动EF`

### 2. 成本管理

#### 挣值分析（EVM）
- **计划值（PV）**：`PV = 计划工作量 × 预算单价`
- **挣值（EV）**：`EV = 已完成工作量 × 预算单价`
- **实际成本（AC）**：实际发生的成本

#### 偏差分析
- **成本偏差（CV）**：`CV = EV - AC`
  - CV > 0：成本节约
  - CV < 0：成本超支

- **进度偏差（SV）**：`SV = EV - PV`
  - SV > 0：进度提前
  - SV < 0：进度落后

#### 绩效指标
- **成本绩效指数（CPI）**：`CPI = EV / AC`
  - CPI > 1：成本节约
  - CPI < 1：成本超支

- **进度绩效指数（SPI）**：`SPI = EV / PV`
  - SPI > 1：进度提前
  - SPI < 1：进度落后

#### 预测
- **完工估算（EAC）**：
  - 按当前效率：`EAC = BAC / CPI`
  - 按计划效率：`EAC = AC + (BAC - EV)`
  - 综合考虑：`EAC = AC + (BAC - EV) / (CPI × SPI)`

- **完工尚需估算（ETC）**：`ETC = EAC - AC`

- **完工尚需绩效指数（TCPI）**：
  - 基于BAC：`TCPI = (BAC - EV) / (BAC - AC)`
  - 基于EAC：`TCPI = (BAC - EV) / (EAC - AC)`

- **完工偏差（VAC）**：`VAC = BAC - EAC`

### 3. 质量管理

#### 标准差与控制图
- **标准差（σ）**：衡量数据离散程度
- **控制上限（UCL）**：`UCL = 均值 + 3σ`
- **控制下限（LCL）**：`LCL = 均值 - 3σ`

### 4. 沟通管理

#### 沟通渠道数
- **沟通渠道数**：`n × (n - 1) / 2`
  - n：项目干系人数量

### 5. 采购管理

#### 合同计算
- **固定总价合同（FFP）**：总价固定
- **成本加固定费用（CPFF）**：`总成本 = 实际成本 + 固定费用`
- **成本加激励费用（CPIF）**：`总成本 = 实际成本 + 目标费用 + 激励费用`
- **成本加百分比费用（CPPC）**：`总成本 = 实际成本 × (1 + 百分比)`

## 📈 常用计算示例

### 示例1：三点估算
某活动乐观时间4天，最可能时间6天，悲观时间14天：
- 期望工期：`TE = (4 + 4×6 + 14) / 6 = 7天`
- 标准差：`σ = (14 - 4) / 6 = 1.67天`

### 示例2：挣值分析
项目预算100万，计划完成50%，实际完成40%，实际花费60万：
- PV = 100 × 50% = 50万
- EV = 100 × 40% = 40万
- AC = 60万
- CV = 40 - 60 = -20万（成本超支）
- SV = 40 - 50 = -10万（进度落后）
- CPI = 40 / 60 = 0.67（成本超支）
- SPI = 40 / 50 = 0.8（进度落后）
- EAC = 100 / 0.67 = 149.25万

### 示例3：沟通渠道
项目有8个干系人：
- 沟通渠道数 = 8 × (8 - 1) / 2 = 28条

## 🎯 考试要点

1. **三点估算**是高频考点，务必掌握公式
2. **挣值分析**是必考内容，要熟练计算各项指标
3. **关键路径**计算要注意正推和反推
4. **沟通渠道**公式简单但容易出错
5. 注意单位换算和小数点精度

## 📝 记忆技巧

- **CV/SV正负判断**：挣值在前为正，在后为负
- **CPI/SPI大小判断**：大于1为好，小于1为差
- **EAC计算**：根据题目条件选择合适公式
- **三点估算**：记住"1-4-1"的权重分配
