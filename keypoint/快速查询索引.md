## 🔢 常用公式索引

### 进度管理公式
- 三点估算：`TE = (To + 4Tm + Tp) / 6`
- 标准差：`σ = (Tp - To) / 6`
- 总浮动时间：`TF = LF - EF = LS - ES`
- 自由浮动时间：`FF = 紧后活动ES最小值 - 本活动EF`

### 成本管理公式
- 成本偏差：`CV = EV - AC`
- 进度偏差：`SV = EV - PV`
- 成本绩效指数：`CPI = EV / AC`
- 进度绩效指数：`SPI = EV / PV`
- 完工估算：`EAC = BAC / CPI`

### 沟通管理公式
- 沟通渠道数：`n × (n - 1) / 2`
