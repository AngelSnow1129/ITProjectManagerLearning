# 📚 信息系统项目管理师学习平台 - 项目信息

## 🎯 项目概述

这是一个专为**信息系统项目管理师**考试打造的在线学习平台，提供完整的学习资源和便捷的学习工具。

### 项目特点
- ✅ **完整内容**：25个章节，覆盖全部考点
- ✅ **三种模式**：完整章节、重点提纲、必背内容
- ✅ **零配置**：双击启动，无需安装
- ✅ **响应式**：支持PC、平板、手机
- ✅ **离线使用**：下载即用，无需网络

## 🚀 快速开始

### 方式1：一键启动（推荐）

**Windows:**
```bash
双击 start_server.bat
```

**Linux/Mac:**
```bash
chmod +x start_server.sh
./start_server.sh
```

### 方式2：手动启动

```bash
python -m http.server 8000
# 访问 http://localhost:8000/web/
```

### 方式3：直接打开

```bash
双击 web/index.html
```

## 🎯 学习路径

### 第一阶段：系统学习（1.5个月）
1. 使用"完整章节"模式
2. 按顺序学习每个章节
3. 重点关注5星章节
4. 做笔记和练习题

### 第二阶段：强化复习（1个月）
1. 使用"重点提纲"模式
2. 重点复习每章核心知识点
3. 做大量练习题
4. 整理错题和难点

### 第三阶段：考前冲刺（0.5个月）
1. 使用"必背内容"模式
2. 快速过一遍所有公式和概念
3. 查漏补缺
4. 模拟考试
