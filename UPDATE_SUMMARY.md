### 对于现有用户

如果您之前使用的是系统架构设计师版本，现在需要：

1. **更新代码**
   ```bash
   git pull origin main
   ```

2. **检查配置**
   - 确认 `web/config.js` 已更新
   - 确认章节文件已更新（01-25章）

3. **清除缓存**
   ```bash
   # 浏览器按 Ctrl+Shift+R 强制刷新
   ```

4. **重新部署**（如果已部署）
   ```bash
   # Cloudflare Pages
   wrangler pages deploy . --project-name=projectmanager-learning
   
   # 或通过Git自动部署
   git push origin main
   ```
