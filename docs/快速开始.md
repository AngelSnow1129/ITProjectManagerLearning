# 🚀 快速开始 - 信息系统项目管理师学习平台

## 📖 三步开始学习

### 第一步：启动平台

选择以下任一方式：

#### 🎯 方式1：一键启动（最简单）

**Windows用户：**
```
双击 start_server.bat
```

**Mac/Linux用户：**
```bash
chmod +x start_server.sh
./start_server.sh
```

✅ 脚本会自动打开浏览器，访问 `http://localhost:8000/web/`

#### 🎯 方式2：直接打开

```
双击 web/index.html
```

⚠️ 注意：某些浏览器可能有跨域限制

#### 🎯 方式3：手动启动服务器

```bash
python -m http.server 8000
```

然后在浏览器访问：`http://localhost:8000/web/`

---

### 第二步：选择学习模式

进入主页后，选择适合您的学习模式：

#### 📖 完整章节
- **适合**：第一次学习、深入理解
- **内容**：完整的教材内容、详细案例
- **章节**：25个完整章节（信息化发展、项目管理、案例分析、论文写作）

#### ⭐ 重点提纲
- **适合**：复习、强化记忆
- **内容**：重难点知识、核心方法、管理工具

#### 🎯 必背内容
- **适合**：考前冲刺、快速查询
- **内容**：核心公式、概念、管理方法
- **章节**：9个核心章节（项目管理十大知识领域）


### 技巧4：离线使用
```
1. 下载整个项目文件夹
2. 复制到U盘
3. 在任何电脑上都可以使用
```

## ❓ 常见问题

### Q1: 页面显示空白？
**A:** 使用启动脚本运行，或手动启动HTTP服务器

### Q2: 图片无法显示？
**A:** 确保 `images` 文件夹在项目目录中

### Q3: 找不到某个章节？
**A:** 使用搜索功能，或查看快速导航

### Q4: 如何打印学习？
**A:** 点击章节页面右上角的"打印"按钮

### Q5: 可以离线使用吗？
**A:** 可以，但需要先加载CDN资源（marked.js和CSS）

## 💪 学习建议

### 时间分配
- **综合知识**：40%（75道选择题，每题1分）
- **案例分析**：30%（3道大题，每题25分）
- **论文写作**：30%（1篇论文，2小时）

### 学习方法
1. **理解为主**：理解项目管理理论和方法
2. **多做练习**：练习案例分析和论文写作
3. **定期复习**：巩固十大知识领域
4. **总结归纳**：建立项目管理知识体系

### 考试技巧
1. **综合知识**：掌握PMBOK知识体系、计算题（EVM、网络图）
2. **案例分析**：分点作答、结合PMBOK理论、条理清晰
3. **论文**：提前准备3-5个项目案例、多练习写作、注意字数和时间
