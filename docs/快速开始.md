## 🎯 平台功能介绍

### 主页导航

进入平台后，您会看到三大学习模块：

#### 1️⃣ 完整章节学习
- **适用场景**：首次学习、系统复习
- **内容特点**：完整教材内容、详细讲解、配图说明
- **推荐用法**：按章节顺序学习，建立完整知识体系
- **包含章节**：25个完整章节

#### 2️⃣ 重点提纲复习
- **适用场景**：二轮复习、强化记忆
- **内容特点**：提炼核心知识点、重难点标注
- **推荐用法**：快速过一遍所有章节，查漏补缺
- **包含章节**：25个精简提纲

#### 3️⃣ 必背内容冲刺
- **适用场景**：考前冲刺、快速查询
- **内容特点**：核心公式、关键概念、必考知识点
- **推荐用法**：考前1-2周强化记忆
- **包含章节**：9个核心章节（十大管理领域）

### 快速工具

在主页下方，您还可以访问：

- **📌 必背索引**：所有必背内容的总索引
- **📐 公式速查**：常用计算公式快速查询
- **🔍 快速查询**：关键知识点索引
- **📊 学习进度**：跟踪您的学习进度

## 📖 如何使用平台学习

### 第一步：选择学习模式

根据您的学习阶段选择合适的模式：

```
首次学习 → 完整章节学习
复习阶段 → 重点提纲复习
考前冲刺 → 必背内容冲刺
```

### 第二步：浏览章节目录

点击任一学习模式后：

1. **查看章节列表**：左侧显示所有章节
2. **星级标注**：⭐⭐⭐⭐⭐ 表示重要程度
3. **搜索功能**：顶部搜索框快速定位章节

### 第三步：学习章节内容

点击章节后：

1. **阅读内容**：右侧显示章节详细内容
2. **查看图表**：支持图片、表格、代码块
3. **做笔记**：建议在纸质笔记本上记录重点
4. **标记进度**：学完后在学习进度表中打勾

### 第四步：使用辅助功能

学习过程中可以：

- **🔍 搜索**：在当前页面搜索关键词
- **🖨️ 打印**：打印当前章节内容
- **⬇️ 下载**：下载Markdown源文件
- **🧭 导航**：使用快速导航跳转

## ❓ 常见问题

### Q1：页面显示空白怎么办？

**原因**：浏览器跨域限制

**解决**：
```
1. 使用启动脚本运行
2. 或手动启动HTTP服务器
3. 不要直接双击HTML文件
```

### Q2：图片无法显示？

**原因**：图片路径错误

**解决**：
```
1. 确保images文件夹存在
2. 检查文件结构是否完整
3. 使用HTTP服务器访问
```

### Q3：如何快速找到某个知识点？

**方法**：
```
1. 使用搜索功能
2. 查看"快速查询索引"
3. 查看"必背内容总索引"
4. 使用浏览器的Ctrl+F搜索
```

### Q4：可以在平板上使用吗？

**可以**：
```
1. 平板连接WiFi
2. 访问电脑的服务器地址
3. 或将项目文件复制到平板
4. 使用平板浏览器打开
```

### Q5：如何跟踪学习进度？

1. 使用"学习进度跟踪表"
