## 📂 目录详细说明

### 1. web/ - 网页文件目录

**用途：** 包含所有网页文件和前端资源

**重要文件：**

| 文件 | 说明 | 访问方式 |
|------|------|----------|
| index.html | 主页，平台入口 | http://localhost:8000/web/ |
| chapters.html | 章节学习页面 | 点击主页的学习按钮 |
| viewer.html | 文档查看器 | 查看索引和速查表 |
| nav.html | 快速导航 | 所有功能快速访问 |
| config.js | 配置文件 | 章节信息管理 |

**路径说明：**
- 网页文件使用相对路径 `../` 访问上级目录的资源
- 例如：`../keypoint/` 访问重点提纲
- 例如：`../06_项目管理概论.md` 访问完整章节

### 2. keypoint/ - 重点提纲和必背内容

**用途：** 存放精心整理的学习资料

**文件类型：**

#### 知识提纲（25个）
- 格式：`XX_章节名_知识提纲.md`
- 内容：重难点知识、公式、方法、案例
- 示例：`10_项目进度管理_知识提纲.md`

#### 必背内容（9个核心章节）
- 格式：`XX_章节名_必背补充.md`
- 内容：核心公式、概念、管理方法
- 章节：06项目管理概论、08整合管理、09范围管理、10进度管理、11成本管理、12质量管理、15风险管理、24案例分析、25论文写作

#### 特殊文档（4个）
- `📌必背内容总索引.md` - 所有必背内容的索引
- `常用公式速查表.md` - 常用公式汇总
- `快速查询索引.md` - 知识点快速查询
- `学习进度跟踪表.md` - 学习进度管理

### 5. 启动脚本

#### start_server.bat (Windows)
- 自动检测Python环境
- 启动HTTP服务器（端口8000）
- 自动打开浏览器访问 `http://localhost:8000/web/`
- 如果没有Python，尝试直接打开 `web/index.html`

#### start_server.sh (Linux/Mac)
- 支持Python3和Python2
- 启动HTTP服务器（端口8000）
- 自动打开浏览器访问 `http://localhost:8000/web/`
- 如果没有Python，尝试直接打开 `web/index.html`

## 🚀 使用流程

### 1. 启动平台

```bash
# Windows
双击 start_server.bat

# Linux/Mac
./start_server.sh

# 或直接打开
双击 web/index.html
```

### 2. 访问主页

- 通过服务器：`http://localhost:8000/web/`
- 直接打开：`web/index.html`

### 3. 选择学习模式

- **完整章节**：查看 `XX_章节名.md`
- **重点提纲**：查看 `keypoint/XX_知识提纲.md`
- **必背内容**：查看 `keypoint/XX_必背补充.md`

### 4. 使用辅助功能

- **搜索**：快速查找章节
- **打印**：打印当前章节
- **下载**：下载Markdown源文件
- **测试**：验证系统功能

## 📝 路径说明

### 相对路径规则

**从web目录访问其他资源：**
- 访问完整章节：`../XX_章节名.md`
- 访问重点提纲：`../keypoint/XX_知识提纲.md`
- 访问必背内容：`../keypoint/XX_必背补充.md`
- 访问图片：`../images/XX_章节名/图片.png`

**示例：**
```javascript
// 在 web/chapters.html 中
const filePath = '../10_项目进度管理.md';  // 访问完整章节
const keypointPath = '../keypoint/10_项目进度管理_知识提纲.md';  // 访问重点提纲
```

### URL参数

**章节学习页面：**
```
chapters.html?type=projectmanager&view=keypoint&chapter=10
```
- `type`: 考试类型（projectmanager）
- `view`: 视图类型（normal/keypoint/mustknow）
- `chapter`: 章节ID（01-25）

**文档查看器：**
```
viewer.html?file=../keypoint/📌必背内容总索引.md
```
- `file`: 文件路径（相对于web目录）

## 🔧 维护说明

### 添加新章节

1. 在根目录添加 `XX_章节名.md`
2. 在 `keypoint/` 添加 `XX_章节名_知识提纲.md`
3. 在 `web/config.js` 中添加章节配置
4. 在 `images/` 创建对应的图片文件夹

### 修改配置

编辑 `web/config.js`：
```javascript
{
    id: '26',
    name: '新章节',
    stars: 4,
    hasKeypoint: true,
    hasMustKnow: false
}
```

### 更新文档

- 主文档：`README.md`
- 快速开始：`docs/快速开始.md`
- 使用演示：`docs/使用演示.md`

## 🎯 总结

这个项目结构的设计原则：

1. **清晰分离**：网页文件和内容文件分开
2. **易于维护**：统一的命名规范和目录结构
3. **灵活访问**：支持多种启动方式
4. **完整文档**：详细的说明和指南

**核心目录：**
- `web/` - 所有网页文件
- `keypoint/` - 重点学习资料
- `images/` - 图片资源
- 根目录 - 完整章节和文档

**启动方式：**
1. 使用启动脚本（推荐）
2. 手动启动服务器
3. 直接打开HTML文件
